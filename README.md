# Student Performance Data Visualization

## Domain
Education Analytics

## Objective
To analyze student academic performance data and identify factors that influence exam results using data visualization techniques.

## Dataset
The dataset contains information about students' study hours, attendance, exam scores, and gender.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Analysis Performed
- Data overview and cleaning
- Exploratory data analysis (EDA)
- Visualization of study habits and academic performance

## Key Insights
- Study hours have a strong positive impact on exam performance.
- Higher attendance is associated with better academic results.

## Future Enhancements
- Build a machine learning model to predict student pass/fail outcomes.
- Deploy insights using a Streamlit dashboard.
